# tibia-economic-analysis
Projeto pessoal de análise de dados focado em controle financeiro, análise temporal e redução de ruído de mercado.

  Objetivo

* Monitorar ganhos diários em moeda do jogo

* Analisar variação de preços de Tibia Coins

* Identificar padrões de lucro por atividade (bosses, hunts, etc)

  Principais técnicas utilizadas

* Média e mediana para análise estatística

* Agregação por data e por mês

* Automação de somatórios condicionais

* Tratamento de outliers (preço com maior volume e menor valor)

  Ferramentas

* Google Sheets / Excel

* Lógica estatística

* Organização de dados temporal
